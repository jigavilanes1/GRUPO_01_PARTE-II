JavaScript HTML DOM

Concepto

El método getElementsByTagName() devuelve un HTMLCollectionobjeto.

Un objeto HTMLCollection es una lista similar a una matriz (colección) de elementos HTML.

const myCollection = document.getElementsByTagName("p");


Se puede acceder a los elementos de la colección mediante un número de índice.

myCollection[1]

Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hola Mundo!</p>
<p>Hola Jacque!</p>
<p id="demo"></p>
<script>
const myCollection = document.getElementsByTagName("p"); document.getElementById("demo").innerHTML = "El innerHTML del segundo parrafo es : " + myCollection[1].innerHTML;
</script> </body>
</html>

Longitud de la Lista

La propiedad length define el número de elementos en un HTMLCollection


myCollection.length


Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hola Mundo!</p>
<p>Hola Jacque!</p>
<p id="demo"></p>
<script>
const myCollection = document.getElementsByTagName("p"); document.getElementById("demo").innerHTML = "Este documento contiene" + myCollection.length + " parrafos.";
</script>
</body>
</html>

La propiedad length es útil cuando desea recorrer los elementos de una colección:

const myCollection = document.getElementsByTagName("p");
for (let i = 0; i < myCollection.length; i++) {
myCollection[i].style.color = "red"; }

Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hola Mundo!</p>
<p>Hola Jacque!</p>
<p id="demo"></p>
<p>Click al boton para cambiar el color de todos los elementos p.</p>
<button onclick="myFunction()">Probar</button> <script>
function myFunction() {
     const myCollection = document.getElementsByTagName("p");
     for (let i = 0; i < myCollection.length; i++) {
          myCollection[i].style.color = "blue";
     }
}
</script>
</body>

Nota

¡Una HTMLCollection NO es una matriz!
Una HTMLCollection puede parecer una matriz, pero no lo es.
Puede recorrer la lista y hacer referencia a los elementos con un número (como una matriz).
Sin embargo, no puede usar métodos de matriz como valueOf(), pop(), push() o join() en una HTMLCollection.
