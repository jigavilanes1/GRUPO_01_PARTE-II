Validación de formulario JavaScript

La validación de formularios HTML se puede realizar mediante JavaScript.

Si un campo de formulario (fname) está vacío, esta función alerta con un mensaje y devuelve falso para evitar que se envíe el formulario:


Ejemplo

<!DOCTYPE html>
<html>
<head>
<script>
function validateForm() {
let x = document.forms["myForm"]["fnamebr"].value;
if (x == "") {
alert("Name must be filled out");
return false;
}else{
alert("Nombre Correcto");
}
</script> </head> <body> <h2>JavaScript Validation</h2> <form name="myForm"
onsubmit="return validateForm()" method="post">
Name: <input type="text" name="fname">
<input type="submit" value="Submit">
</form> </body>
</html>


Validación JavaScript

Nombre:

JavaScript puede validar la entrada numérica

Para cambiar el valor de un atributo HTML, use esta sintaxis:

document.getElementById(id).attribute = new value


Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>Validación JavaScript</h2>
<p>Ingrese un número entre 1 a 10:</p>
<input id="numb">
<button type="button" onclick="myFunction()">Enviar</button>
<p id="demo"></p>
<script>
function myFunction() {
let x = document.getElementById("numb").value;
let text;
if (isNaN(x) || x < 1 || x > 10) {
text = "Dato no Valido";
} else {
text = "Dato Valido";
}
document.getElementById("demo").innerHTML = text;
}
</script>
</body>
</html>
